To run site locally:
  Download MAMP (Mac) or some program that allows you run LAMP stack web applications
  For MAMP:
    Initialize Git repo in directory of your choice
    Run MAMP
    MAMP->Preferences->Web-Server
    Select the Git repo as the Document Root
    Start Servers through MAMP
    Click 'Open WebStart page'
    Select 'My Website' in the top navigation
